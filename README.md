# Portfólio Front-End Simples

Este é um projeto simples de front-end, criado para praticar HTML e CSS.

## Tecnologias utilizadas
- HTML5
- CSS3

## Objetivo
Criar uma página de apresentação pessoal para portfólio, exibindo:
- Sobre mim
- Habilidades
- Projetos
- Contato com botões elegantes

## Visual
O site utiliza tons marrons e bege para uma aparência elegante e atraente.
